2. Charge $5 via Stripe for each new employee added
Goal: Add a $5 one-time fee every time someone clicks “Add Staff” and successfully creates a new team member.
Recommended implementation path
Step 1 – Decide on the exact trigger
Most logical moment:
Charge after the user clicks “Add Staff” in the modal and the employee is successfully saved to the database.
Step 2 – Create a $5 one-time product/price in Stripe

Go to your Stripe Dashboard → Products → + Create product
Name: “New Employee Add-on Fee”
Pricing: One-time fee → $5.00 USD
Save → note the Price ID (looks like price_1xxxxxxxxxxxxx)

Step 3 – Add Stripe checkout in your app
Two clean ways to do it:
Way A – Redirect to Stripe Checkout (easiest & most secure)
When user clicks “Add Staff” and the employee is created successfully:

Create a Stripe Checkout Session on your backend
Redirect user to Stripe to pay $5
After successful payment → redirect back to your app and show “Employee added”

Backend flow (Node.js example):
JavaScript// POST /api/add-employee
async function addEmployee(req, res) {
  // 1. Save the employee to your database first
  const newEmployee = await db.staff.create({ ...req.body });

  // 2. Create Stripe Checkout Session
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [{
      price: 'price_xxxxxxxxxxxxxxx', // your $5 price ID
      quantity: 1,
    }],
    mode: 'payment',
    success_url: `${YOUR_APP_URL}/staff?success=true&employeeId=${newEmployee.id}`,
    cancel_url: `${YOUR_APP_URL}/staff?cancel=true`,
    metadata: { employeeId: newEmployee.id }
  });

  // 3. Redirect user to Stripe
  res.json({ url: session.url });
}
