AI TOOL: Social Media Post Builder
“Content Library + Guided Prompts + Holiday Suggestions”
Purpose

Create consistent, on-brand posts fast—without hunting for photos or reinventing what to say. This tool turns:

your photo library +

a few key inputs +

upcoming relevant “national day” prompts
into finished posts for IG/FB (and optionally Stories/Reels captions).

Step 4 — Holiday Suggestion (Today + Next 5 Days)

Each holiday entry includes:

Controls:

Important note (quality control):
National “day” sources vary. Your tool should:

Use a consistent source

Allow an override (“Custom holiday”)

Avoid weird or off-brand holidays by default

Step 5 — Generate Content (Outputs)

One click: Generate Post.

AI returns:

A) Primary Caption (platform-optimized length)
B) Short Caption Variant
C) Story Text Overlay Suggestions (3 options)
D) Hashtags (3–8, local + category)
E) Posting Time Suggestion (based on your best windows)
F) Optional Reply Pack (2–3 quick responses to comments)

Posting Time Engine (simple but effective):

Weekdays: 10:30–11:30 AM and 4:30–5:30 PM

Weekends: 9:00–10:30 AM and 4:00–5:00 PM

“If event is tonight” → pushes to afternoon window automatically

5) Brand Voice Settings (Owner Controls)

A settings page that lets you lock standards:

Voice adjectives: warm / confident / welcoming / premium / playful

Default CTA: reserve / walk-ins / order online

“Never say” list (e.g., “cheap”, “best ever”, “guaranteed”)

Emoji level: none / light / normal

Hashtag style: none / minimal / local-focused

This makes posts consistent even if a manager generates them.
